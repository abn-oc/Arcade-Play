Online realtime gaming platform, Currently has Tic tac toe.

## Tech Stack:
Frontend: React
Backend: Node.js + Express (used JWT and OAuth for Google sign-in)
Database: PostgreSQL (started with MSSQL and switched later)
Sockets.io for Realtime Communication
